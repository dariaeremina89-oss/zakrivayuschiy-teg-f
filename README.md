# Закрывающий тег

Учебный проект Яндекс Практикума по HTML и CSS.

Проект посвящён адаптивной верстке и работе с современными возможностями CSS.

## Что реализовано

- адаптивная и резиновая верстка по макету Figma;
- CSS-переменные;
- вариативный шрифт;
- сложный фон на основе нескольких градиентов;
- CSS-фильтры для изображений;
- анимации SVG-иконки лайка;
- интерактивные состояния кнопок;
- модальное окно на элементе `dialog`.

## Технологии

- HTML5
- CSS3
- SVG
